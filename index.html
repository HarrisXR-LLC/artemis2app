About this app:

This app was created by Aaron Harris on April 1st 2026.

After the Artemis II launch, I had some difficulty in finding an at-a-glance and quick-to-open timeline view for Artemis II and its mission path and so this web page app was made to be a simplified illustrative view of the timeline.

This app is not meant to be highly scientifically accurate, but rather just a general locator of the mission and upcoming events. I did my best to make sure everything lined up for times and scales.

The app was created using Claude Code (Opus 4.6) with Codex 5.4 as an "investigator", and deliberately made a monofile (all-in-one) HTML CSS JS page so there were no outside dependencies. One file has it all.

Released "as-is" under the MIT License.
